(ENG)

Auto Sorting Files

A Python-based utility that automatically monitors your Downloads folder and sorts files into specific categories on a different drive.

Requirements:

    Create the following folders on your D: drive: Video, Music, Photo, Archives.

    In the sort_files.py code, update the SOURCE_DIR variable with your actual path, for example:
    SOURCE_DIR = "C:/Users/YourName/Downloads"

Usage:

    Run via CMD: python sort_files.py (or use mysort.bat).

(RUS)

Автоматическое сортирование файлов

Скрипт на Python, который автоматически сканирует папку «Загрузки» и распределяет файлы по категориям на другом диске.

Требования:

    Создайте на диске D: папки: Video, Music, Photo, Archives.

    В коде sort_files.py укажите свой путь к загрузкам в переменной SOURCE_DIR, например:
    SOURCE_DIR = "C:/Users/ИмяПользователя/Downloads"

Использование:

    Запустите через командную строку: python sort_files.py (или через mysort.bat).
